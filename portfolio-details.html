Click Libro - Página web de venta de libros

Click Libro es un proyecto desarrollado como página web de comercio electrónico para la compra y gestión de libros en línea. El sistema permite a los usuarios registrarse, iniciar sesión, explorar un catálogo de libros, añadir productos al carrito y generar pedidos. Incluye módulos de usuario y administrador para la gestión de inventario y pedidos, ofreciendo una experiencia de compra sencilla y funcional.
